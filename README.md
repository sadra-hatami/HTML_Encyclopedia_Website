## 📖 About

HTML Encyclopedia is a comprehensive educational website created to serve as a complete reference for HTML.

Unlike traditional tutorials, this project focuses on documenting the entire HTML ecosystem—from commonly used elements to deprecated and obsolete tags—making it suitable for both beginners and experienced developers.

Each documented element includes:

* Detailed explanations
* Supported attributes
* Practical code examples
* Usage notes
* Best practices
